# Athlete Recovery Prediction System

## Project Overview

This project predicts an athlete's recovery score using Machine Learning.

The system includes:

- Data preprocessing
- Data validation using Great Expectations
- Random Forest Regression model
- FastAPI REST API
- PostgreSQL database
- Streamlit web application
- Interactive dashboard
- Batch prediction using CSV files
- Docker containerization
